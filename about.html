About This Website

The purpose of this website is to showcase the rich film heritage of Arkansas. Many well-known movies have been filmed in Arkansas, and many successful actors call the Natural State home.

This website was developed using:

  • HTML, CSS, and JavaScript
  • The TMDB API for movie and actor data
  • Encyclopedia of Arkansas for Arkansas-specific research
  • Google Analytics to analyze visitor usage
  • GitHub Pages for hosting

This project is intended for audiences aged 20–40 who are interested in film, culture, and Arkansas history.

All movie posters, actor images, and filmographies are sourced dynamically using the TMDB API.
